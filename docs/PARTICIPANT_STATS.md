# Participant Stats

Participant stats are aggregated from:

- destination metric rows;
- campaign performance rows;
- publishing tasks and final URLs;
- participant submissions;
- participant-destination links.

Primary mapping:

```text
posted_url
-> publishing_task
-> destination
-> participant link
-> SKU/content run/variant/campaign
-> metrics
-> participant dashboard
```

The dashboard reports assignments, submissions, approval rate, published count, views, clicks, orders, revenue, engagement rate, and payout totals.
