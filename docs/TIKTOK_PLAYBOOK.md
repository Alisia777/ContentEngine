# TikTok Playbook

TikTok tasks must stay inside authorized paths.

Allowed:

- official connector when authorized;
- CSV/manual report fallback;
- tracking link for clicks;
- final URL for post identity.

Not allowed:

- unofficial login;
- scraping private pages;
- private API;
- proxy or anti-detect bypass;
- silent fallback that pretends data was collected.

Return views, likes, comments, shares, saves when available, and clicks via tracking link.
