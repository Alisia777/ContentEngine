# Account Setup Checklist

Destination setup tasks use a checklist to keep account setup explicit and reviewable.

Required operator checks:

- create or confirm an owned account off-platform;
- confirm platform rules and brand ownership policy;
- apply approved name, handle, avatar, bio, and official link;
- assign a human owner;
- confirm only approved videos can be published;
- connect official OAuth/API token when available;
- use manual-assisted upload when API is unavailable or token is not valid;
- record final handle or URL before marking setup complete.

ContentEngine does not perform external account registration. The final internal destination is created only after an operator supplies a final URL or handle.
