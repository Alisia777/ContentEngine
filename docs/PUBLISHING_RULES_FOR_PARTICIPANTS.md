# Publishing Rules For Participants

## Before Publishing

- Video status is approved.
- Assignment exists.
- Destination is linked to the participant.
- Caption, title, description, hashtags, CTA, SKU, and payout rule are present.
- Tracking link exists for the publishing task.

## During Publishing

Use the tracking link in the post description, caption, comment, or link field.

Correct:

```text
Learn more: https://our-domain.com/r/abc123
```

Incorrect:

```text
Learn more: https://ozon.ru/direct-product-url
```

Direct product links hide clicks from ContentEngine.

## After Publishing

Submit the final platform URL:

```text
final_url = https://facebook.com/...
```

The task is incomplete until `final_url` is saved.
