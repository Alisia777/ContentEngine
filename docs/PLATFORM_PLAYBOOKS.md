# Platform Playbooks

Training Academy includes platform-specific onboarding because every network returns different evidence and metrics.

Core questions each playbook answers:

1. What do I do on this platform?
2. What data must I return?
3. What happens if I submit links or stats incorrectly?

Platform readiness is based on certification badges. By default gates are advisory; service-level strict gates can block assignment or report submission when enabled.

Playbooks:

- Instagram / Reels
- Facebook
- YouTube Shorts
- TikTok
- Telegram
- VK
- Ozon / Wildberries / marketplace metrics
- Partner slots

Universal rules:

- Work only through assignment.
- Publish only approved video.
- Use `tracking_link`.
- Submit `final_url`.
- Submit stats with `posted_url`, `tracking_slug`, or `publishing_task_id`.
- Do not use unofficial login, scraping, private APIs, proxies, or permission bypasses.
