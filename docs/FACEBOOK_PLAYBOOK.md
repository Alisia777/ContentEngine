# Facebook Playbook

Inside ContentEngine, Facebook work is tied to a destination: page, profile, partner slot, or operator-managed placement.

Use official Meta connection when available. Otherwise use CSV/manual fallback.

Return:

- `final_url`
- `tracking_link` evidence
- `posted_url` or `tracking_slug`
- views
- reach
- impressions
- likes
- comments
- shares
- clicks

Stats connect to payouts only after attribution by final URL, tracking slug, publishing task, or approved manual mapping.
