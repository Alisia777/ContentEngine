# Destination Profile Packs

Destination profile packs are internal setup briefs for owned campaign destinations.

Each pack includes:

- campaign and platform;
- SKU focus;
- suggested account name and handle;
- safe bio text;
- avatar prompt or existing asset path;
- content pillars;
- first 9 post ideas;
- posting rules.

The first 9 posts are planning ideas only. They do not create videos, do not publish content, and do not bypass human review. A publishing task can only be scheduled against an internal destination after the operator completes setup and the destination is created.

For API-capable platforms, posting rules prefer official API upload when OAuth/token status is valid. For other platforms, the workflow stays manual-assisted and records the final URL after publication.
