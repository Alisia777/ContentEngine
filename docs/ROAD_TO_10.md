# Road to 10/10

Road to 10/10 is generated from EngineAudit recommendations and surfaced in Control Room.

The rule is simple:

1. `/engine-audit` measures what is weak.
2. `/control-room` shows what each role should do today.
3. Gated actions remain gated.
4. Paid smoke stays blocked until credits, role, spend gate, scene policy, and OutputAcceptance are ready.

After v3.5, the operating order remains:

1. Confirm Runway credits.
2. Run exactly one paid smoke.
3. Review through OutputAcceptance.
4. If product identity fails, implement product compositing / packshot overlay.
5. If product passes, save it as a benchmark candidate.
6. Then pilot 5 SKU.
