Unified Control Room v3.5

ContentEngine кабинеты и оболочка запуска

Это публичная статическая витрина GitHub Pages. Рабочие кабинеты, данные, роли, EngineAudit и очереди действий запускаются локально через FastAPI.

Local app

Как открыть настоящие кабинеты

Control Roomlocalhost:8013
Owner dashboard/control-room?role=owner
ALTEA motion shell/altea-motion/dashboard

GitHub Pages не запускает Python/FastAPI. Поэтому здесь лежит entry shell, а операционный слой открывается после локального запуска сервера.

Cabinets

Ролевые рабочие экраны

Эти маршруты доступны в локальной версии приложения.

Owner

Executive Control Room

Production readiness, EngineAudit score, blockers, paid smoke status, decisions.

 Content lead

Quality & Briefs

Product strategy, weak briefs, one-video plans, prompt-only readiness, review queue.

 Reviewer

Output Acceptance

Human review, product identity, packaging drift, regeneration and QA decisions.

 Creator

Assignments & Payouts

Brief cards, final URL blockers, stats, training and payout status.

 Metrics

Metrics Intake

Unmatched rows, missing stats, destinations without stats, funnel coverage.

 Shell

ALTEA Motion UI

Premium visual shell: gold ribbons, petals, dashboard motion and loading layers.

Run locally

Команды запуска

Из корня репозитория: python -m uvicorn app.main:app --host 127.0.0.1 --port 8013. После запуска открывай http://localhost:8013/control-room?role=owner.
